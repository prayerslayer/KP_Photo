Lernsoftware Panoramaerstellung

Bilder laden

Im ersten Schritt der Panoramaerstellung werden die einzelnen Teilbilder geladen. Professioneller Software ist es egal, ob diese Teilbilder ein horizontales oder vertikales (oder beides) Panorama beschreiben und in welcher Reihenfolge diese angegeben werden. In dieser Lernsoftware müssen Bilder von links nach rechts ein horizontales Panorama beschreiben. Das bedeutet das erste Bild ist das linkeste Teilbild des Panoramas, das Zweite das zweitlinkeste etc. Bezüglich notwendiger Überlappung wurden keine Tests durchgeführt, weswegen keine Aussage dazu getroffen werden kann.

Bedienung

Mit dem Button Eigene Bilder laden können eigene Teilbilder eines Panoramas geladen werden. Dabei sind oben beschriebene Einschränkungen zu beachten, ansonsten kann kein Panorama erstellt werden. Der Button Standardbilder laden lädt drei Bilder mit Blick auf das Institutsgebäude. Da mit diesen die Software getestet wurde, ist ein positives Ergebnis garantiert. Wurde ein Bild versehentlich geladen und soll gelöscht werden, ist das mit dem Button (Mülleimer-Icon) über dem Bild möglich. Leider können Bilder nicht neu angeordnet werden, sobald sie einmal geladen wurden.
